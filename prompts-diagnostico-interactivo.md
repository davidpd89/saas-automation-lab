# Prompts para mejorar el diagnostico interactivo

Fecha: 2026-05-08

## 1. UX de diagnostico para web afiliada B2B

Analiza patrones de diagnostico interactivo en webs B2B, SaaS y afiliacion tecnica. Busca ejemplos actuales de 2026 donde una home pregunta al usuario su problema y lo dirige a una solucion. Extrae estructura, numero optimo de preguntas, microcopy, errores de UX y medicion de conversion.

## 2. Chat guiado sin IA vs chatbot generativo

Compara un diagnostico guiado por botones frente a un chatbot generativo para una web estatica de afiliacion SaaS. Evalua impacto en confianza, Core Web Vitals, privacidad, mantenimiento, legalidad, AI Act y conversion.

## 3. Preguntas de cualificacion para coaches

Disena un flujo de 5 preguntas para coaches que quieren automatizar captacion, onboarding, WhatsApp, pagos y seguimiento. Cada respuesta debe mapearse a una guia, herramienta o comparativa de SaaS Automation Lab.

## 4. Preguntas de cualificacion para formadores

Disena un flujo de 5 preguntas para formadores y creadores de cursos. Debe distinguir entre entrega de curso, email marketing, evergreen, comunidad, soporte y analitica. Devuelve rutas recomendadas y CTAs.

## 5. Preguntas de cualificacion para consultores B2B

Disena un flujo de 5 preguntas para consultores B2B que quieren automatizar prospeccion, CRM, propuestas, facturacion y soporte. Incluye umbrales de volumen y nivel tecnico para recomendar n8n, Make, Zapier, Clientify, Pipedrive o lemlist.

## 6. Copy de respuestas del asistente

Escribe 30 respuestas breves de un asistente editorial para SaaS Automation Lab. Tono: claro, directo, no agresivo, experto pero no vendedor. Cada respuesta debe orientar a una decision y mencionar una limitacion.

## 7. Medicion de eventos del diagnostico

Propone un esquema de tracking privacy-first para medir el diagnostico: eventos, propiedades, almacenamiento local, GA4 opcional, Plausible opcional, conversiones y calidad del click afiliado.

## 8. SEO y AEO del diagnostico

Investiga si un diagnostico interactivo en home ayuda o perjudica SEO/AEO. Propone como hacerlo indexable: texto visible, FAQ, schema, enlaces internos, no esconder contenido importante tras JS y compatibilidad con lectores de pantalla.

## 9. Personalizacion por problema

Define una matriz de recomendacion para SaaS Automation Lab con columnas: problema, perfil, nivel tecnico, volumen mensual, herramientas candidatas, primera pagina recomendada, CTA secundario y contenido faltante.

## 10. CRO de CTAs despues del diagnostico

Investiga mejores CTAs para usuarios que acaban de completar un diagnostico B2B. Compara "ver guia", "comparar herramientas", "calcular ahorro", "descargar plantilla" e "ir a web oficial". Prioriza confianza frente a afiliacion agresiva.

## 11. Riesgos legales de un asistente de recomendacion

Analiza riesgos legales y editoriales de un asistente que recomienda herramientas SaaS afiliadas en Espana/UE en 2026. Incluye disclosure, consentimiento, privacidad, AI Act, sesgos, logs y avisos.

## 12. Benchmark de paginas tipo quiz

Busca ejemplos de quizzes o diagnosticadores usados para lead generation en SaaS, consultoria y herramientas no-code. Resume estructura, longitud, campos de email, incentivo y tasa de conversion si hay datos publicos.

## 13. Diagnostico para comparativa n8n vs Make vs Zapier

Crea un arbol de decision especifico para recomendar n8n, Make o Zapier segun volumen, nivel tecnico, privacidad, coste, necesidad de self-hosting, integraciones y mantenimiento.

## 14. Diagnostico para stack de email y cursos

Crea un arbol de decision para elegir entre Kit, MailerLite, Brevo y Thinkific segun tipo de producto, lista actual, automatizaciones, presupuesto y equipo.

## 15. Futuras herramientas interactivas

Propone 20 utilidades interactivas para SaaS Automation Lab: calculadoras, checklists, comparadores, generadores de stack, plantillas y mini demos. Para cada una indica objetivo SEO, dificultad y pagina donde encaja.
